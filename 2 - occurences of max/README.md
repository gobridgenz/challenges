# Number of occurrences of max

Count the number of occurrences of the maximum number in a list.

### Example 1

```
[3 0 1 3]
```

Should print:

```
2
```

#### Explanation:

max number = 3 and appears 2 times


### Example 2

```
[18 90 90 13 90 75 90 8 90 43]
```

Should print:

```
5
```

#### Explanation:

max number = 90 and appears 5 times
