# Christmas tree

Print a christmas tree which can scale to a given height.

Feel free to build your tree the way you want! Just remember that it should have a trunk in the middle and as many rows as asked.

### Example 1:

#### Height of 10

```
                    o                    
                   ooo                   
                  ooooo                  
                 ooooooo                 
                ooooooooo                
               ooooooooooo               
              ooooooooooooo              
             ooooooooooooooo             
            ooooooooooooooooo            
           ooooooooooooooooooo           
                    "
```

#### Height of 5

```
          o          
         ooo         
        ooooo        
       ooooooo       
      ooooooooo      
          "      
```
